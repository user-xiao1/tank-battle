# 坦克大战 (Tank Battle)

一个使用纯 HTML5、CSS3 和原生 JavaScript 实现的经典坦克大战小游戏。

## 🎮 在线试玩

直接在浏览器中打开 `index.html` 即可开始游戏。

## 📋 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格` | 发射子弹 |

## 🚀 游戏目标

- 控制青色坦克，消灭所有红色敌方坦克
- 保护基地（红色方块）不被摧毁
- 注意自己的生命值，生命值耗尽则游戏失败

## ✨ 游戏特性

- 🟦 玩家坦克：流畅移动与射击
- 🔴 敌方坦克：简单 AI 自动巡逻与攻击
- 🧱 砖墙与钢墙：不同墙体有不同耐久
- 💥 粒子爆炸效果
- 📱 基础触屏支持（滑动移动 + 自动射击）
- 🏆 胜利/失败结算界面

## 🛠 技术栈

- HTML5 Canvas
- CSS3
- Vanilla JavaScript（无第三方依赖）

## 📦 本地运行

```bash
# 克隆仓库后，直接用浏览器打开
double-click index.html

# 或者使用本地服务器
python -m http.server 8080
```
